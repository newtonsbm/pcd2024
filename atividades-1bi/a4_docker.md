# Atividade A4 Docker

- Rode um CMS ou plataforma de blog em um container, faça o mapeamento de portas e as configurações necessárias para acessá-la pelo navegador no hospedeiro
- Crie uma pasta nova no github de atividades chamada a4_docker
- Coloque 1 print da plataforma sendo acessada pelo navegador do host na pasta a04_docker criada
- Exemplos:
    - Plataformas
    - Wordpress
    - Ghost Blog
    - Joomla
    - Drupal
    - ou alguma a sua escolha
- https://hub.docker.com/search
